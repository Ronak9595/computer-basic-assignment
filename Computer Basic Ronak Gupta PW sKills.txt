1. What is a computer?
=> A computer is an electronic device which accepts input from input deveice, process the data the data using processer and displays the results using output device.

2. What is a RAM?
=> A RAM is a primary memory also called as main memory which is directly connected to processer. It is volatile in nature means instruction will be stored until the power is supplying and instruction will be lost when power is not supplying.

3. Where is data stored in computer?
=> Data is stored in secondary memory which are HDD and SDD.
